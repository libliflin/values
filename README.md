# values

A manifesto on value-driven development.

→ **[libliflin.github.io/values](https://libliflin.github.io/values)**

The thesis: agents are smart enough now that we should not be giving them
specs. We should be giving them values, personas, and themes, and they can
figure out the rest.

[Lathe](https://github.com/libliflin/lathe) is the concrete implementation.
